安装onos控制器的方法

Introduction

根据官方的教程走，然后会发现，what！！！咋能安装不上呢？不得不说这玩意真的难安装（对于国内开发者而言），踩了不少的坑，然后终于在师兄帮助下才成功安装，因此，写篇文章记录一下，希望也能够帮助使用onos进行开发的开发者们。

Step1

安装Bazel和其它的依赖项（使用的操作系统为Ubuntu18.04）
Bazel的官方安装方法

注意：不要安装最新的Bazel，因为，bazel版本迭代很快，onos可能还没有支持

The minimum required version of Bazel at the time of writing is 1.0.

官网上说bazel版本至少为1.0，这里我安装的是1.2.0版本

建议把JDK安装上,虽然官方说不用安装

Do I need to install a JDK? The short answer is NO (smile)

JDK安装方法

替他的依赖项可以直接用apt install 进行安装

git clone https://gerrit.onosproject.org/onos
cd onos
bazel build onos

export HTTPS_PROXY=https://:
export HTTP_PROXY=http://:

bazel build onos –action_env=HTTP_PROXY=$HTTP_PROXY

bazel run onos-local – clean debug

‘clean’ to delete all previous running status

‘debug’ to enable remote debugging

tools/test/bin/onos localhost

onos> app activate org.onosproject.openflow
onos> app activate org.onosproject.fwd

sudo mn –controller remote,ip= –topo torus,3,3
```
这里的填写127.0.0.1（不要直接输入localhost）就行了，因为onos是默认运行在本机的还回地址上的

End

好了，该到说再见的时候了，希望大家能够安装顺利。
我也得熟悉下onos的使用了，毕竟项目需要···不得不顶住呐~
